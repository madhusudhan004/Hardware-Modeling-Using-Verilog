# Hardware-Modeling-Using-Verilog
Hello Madhusudhan K V here,
This Repository consists of verilog code or programs that is executed during the 45 Days Verilog practice sessions. It contains programs and solutions for digital electronics problems solved using Verilog HDL Language. Concepts include on basic digital electronics modules such as gates, latches, flip flops, Counters, Multiplexer.

Requirements-
1) Icarus Verilog
2) GTKwave
3) OSS CAD Suite-Yosys

Icarus verilog is used to compile and debug verilog programs. The simulation is observed in GTKwave wave visualizer.
Yosys is a open source RTL Synthesis software. The verilog designs are converted to RTL netlist using TCL of Yosys.
Synthesis files,Images and netlist files uploaded are synthesized using the Yosys open source platform.
